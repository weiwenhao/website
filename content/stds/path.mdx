# path

## Overview
The `path` standard library provides basic functionalities for handling file paths, including operations such as path joining, path splitting, and path checking.

## Functions

### exists
Checks if a file or directory exists.

```nature
fn exists(string path):bool
```

### isdir
Checks if the specified path is a directory.

```nature
fn isdir(string path):bool
```

### base
Gets the last element of the path (filename or directory name).

```nature
fn base(string path):string
```

Example:
```nature
println(path.base("/foo/bar/baz.txt"))  // Output: baz.txt
println(path.base("/foo/bar/"))         // Output: bar
println(path.base(""))                  // Output: .
```

### dir
Gets the directory portion of the path.

```nature
fn dir(string path):string
```

Example:
```nature
println(path.dir("/foo/bar/baz.txt"))  // Output: /foo/bar
println(path.dir("/foo/bar/"))         // Output: /foo/bar
println(path.dir("/foo"))              // Output: /
println(path.dir(""))                  // Output: .
```

### join
Joins multiple path elements into a single path.

```nature
fn join(string dst, ...[string] list):string
```

Example:
```nature
import path

fn main() {
    // Basic path operations
    var filepath = "/foo/bar/baz.txt"
    println(path.base(filepath))    // Output: baz.txt
    println(path.dir(filepath))     // Output: /foo/bar

    // Path joining
    println(path.join("/foo", "bar", "baz.txt"))  // Output: /foo/bar/baz.txt

    // Path checking
    println(path.exists("/etc"))           // Check if path exists
    println(path.isdir("/etc"))            // Check if it is a directory
}
```

## Notes
1. The `join()` function treats paths starting with `/` as absolute paths.
2. The path separator is uniformly `/`.
3. Special handling of empty paths:
   - `base("")` returns ".".
   - `dir("")` returns ".".
4. A trailing `/` at the end of a path is automatically handled in certain operations.
