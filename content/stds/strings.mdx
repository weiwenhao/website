# strings

## Overview
The `strings` standard library provides a series of basic string manipulation functionalities, including common operations such as string searching, splitting, replacing, and trimming.

## Functions

### from
Creates a Nature string based on a C string.

```nature
fn from(anyptr p):string!
```

### join
Joins a string array into a single string using the specified separator.

```nature
fn join([string] list, string separator):string
```

Example:
```nature
var list = ["hello", "world"]
println(strings.join(list, " "))  // Output: hello world
```

## string Methods

### find
Finds a substring, returns the position of the first occurrence, returns -1 if not found.

```nature
fn string.find(string sub):int
```

### find_after
Finds a substring starting from the specified position.

```nature
fn string.find_after(string sub, int after):int
```

### rfind
Finds a substring from right to left.

```nature
fn string.rfind(string sub, int before):int
```

### contains
Checks if the string contains the specified substring.

```nature
fn string.contains(string sub):bool
```

### starts_with
Checks if the string starts with the specified substring.

```nature
fn string.starts_with(string starts):bool
```

### ends_with
Checks if the string ends with the specified substring.

```nature
fn string.ends_with(string ends):bool
```

### split
Splits the string using the specified separator.

```nature
fn string.split(string separator):[string]
```

### slice
Extracts a specified portion of the string.

```nature
fn string.slice(int start, int end):string
```

### trim
Removes the specified set of characters from both ends of the string.

```nature
fn string.trim([string] list):string
```

### ltrim
Removes the specified set of characters from the left end of the string.

```nature
fn string.ltrim([string] list):string
```

### rtrim
Removes the specified set of characters from the right end of the string.

```nature
fn string.rtrim([string] list):string
```

### replace
Replaces the specified substring within the string.

```nature
fn string.replace(string sub_old, string sub_new):string
```

### reverse
Reverses the string.

```nature
fn string.reverse():string
```

### to_int
Converts the string to an integer.

```nature
fn string.to_int():int!
```

### to_float

```nature
fn string.to_float():float!
```

### ascii
Gets the ASCII code of the first character of the string.

```nature
fn string.ascii():u8
```

## Usage Example

```nature
import strings

fn main() {
    var str = "Hello, World!"

    // String searching
    println(str.find("World"))      // Output: 7
    println(str.contains("Hello"))  // Output: true

    // String splitting
    var parts = str.split(", ")
    println(parts)                  // Output: ["Hello", "World!"]

    // String trimming
    var text = "  hello  "
    println(text.trim([" "]))       // Output: "hello"

    // String replacing
    println(str.replace("World", "Nature"))  // Output: Hello, Nature!

    // String conversion
    var num = "123".to_int()
    println(num)                  // Output: 123
}
```

## Notes
1. The `to_int()` and `to_float()` methods will throw an exception on conversion failure.
2. `find()` related methods return -1 when not found.
3. String indexing starts from 0.
4. Strings are immutable; all modification operations return a new string.
