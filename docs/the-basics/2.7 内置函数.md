内置函数是不需要通过 import 等语句可以直接调用的全局函数

## print/println

定义

```nature
fn print(...[any] args) {}

fn println(...[any] args) {}
```


仅支持 bool/string/number 类型数据进行打印。println 相比于 print 多了一个 `\n`
示例 `println("hello", true, "world")`

## set

定义

```
fn set(...[T] keys):{T} {}
```

初始化一个 set 结构数据，示例 `var s = set()` 或者 `var s = set(1, 2, 3)`
